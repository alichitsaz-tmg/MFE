Firmware Updates

Keep your launch monitors up to date for best performance

GCQuad / QuadMAX (Foresight)

Update via PC (Windows)

  1. Download the firmware updater from Foresight
  2. Power on your GCQuad / QuadMAX
  3. Connect to PC via USB-C cable
  4. Run the firmware installer (.exe)
  5. Click Update when prompted
  6. Wait for completion — device may restart

  1. Download the Foresight App from App Store
  2. Power on your GCQuad / QuadMAX

TrackMan 4

Update via Web Interface

  1. Download the firmware (.bin file) from TrackMan Support
  2. Connect TrackMan 4 to PC via Ethernet or Wi-Fi
  3. Open TrackMan Performance Studio (TPS)
  4. Go to Settings → Advanced → About → Show Device Web Interface
  5. Click Login (no password required)
  6. Navigate to Administration → Upgrade
  7. Select the .bin firmware file and click Upgrade
  8. Wait for restart — green light indicates completion

Experiential Team Troubleshooting

Common issues and solutions for iPad users

Monitor Not Connecting

Check that you're connected to the monitor's Wi-Fi network, not venue Wi-Fi or your regular network.

iPad Settings → Wi-Fi → Select TrackMan_#### or GCQuad_####

App Crashing

Check if iOS was recently updated. New iOS versions may need an app update. Force close and relaunch first.

If persists after iOS update → Contact TaylorMade IT

Can't Login

Verify your credentials and check CAPS LOCK. Account locked after too many attempts?

Password reset → Call 800-864-7230

Data Not Syncing

Fittings done offline will sync when back online. Open the app with internet connection to trigger sync.

Check Settings → Sync status indicator

Retail Partners Troubleshooting

Common issues and solutions for Windows users

Monitor Not Connecting

Check that Connector Service is installed and running (system tray icon). Verify USB cable or network connection.

Open Services (services.msc) → Find "TaylorMade Connector Service" → Start

Connector Issues

If Connector won't install or start, you may have denied the UAC prompt or lack admin rights.

Uninstall from Control Panel → Reinstall → Click YES on UAC prompt

App Won't Launch

Windows SmartScreen may be blocking the app. Check Event Viewer for crash details.

Click "More info" → "Run anyway" if blocked by SmartScreen

Can't Login

Retail accounts are different from internal My Fitting EXP accounts. Check for activation email.

Password reset → Contact TaylorMade IT

Complete Reinstall

If issues persist, do a clean reinstall of both MFE and Connector Service.

Settings → Apps → Uninstall MFE → Control Panel → Uninstall Connector → Reboot → Reinstall both
