INSTALLATION GUIDE

Follow these steps to install MFE on your Windows device

‚ö†Ô∏è ADMIN RIGHTS REQUIRED You will need administrator approval during the Connector Service installation. Have your IT team contact information ready before beginning.
üì• Download MFE from Microsoft Store üîå Download Connector Service
1

Install MFE from Microsoft Store

  1. Click the download link: MFE on Microsoft Store
  2. On the Microsoft Store page, click Get or Install
  3. If prompted, sign in with your Microsoft account
  4. Wait for the installation to complete
2

Install Connector Service (Admin Rights Required)

üîë THIS STEP REQUIRES ADMIN APPROVAL The Connector Service enables communication between MFE and your launch monitor
  1. Download the Connector: TaylorMade Connector Service
  2. Run the installer executable file
  3. When you see a UAC (User Account Control) prompt, click YES to continue
  4. If you see a password prompt, contact TaylorMade IT immediately at 800-864-7230
  5. Wait for the Connector installation to complete
3

Restart MFE Application

  • After installing the Connector, close MFE completely
  • Reopen the MFE app from your Start Menu or desktop
  • The Connector service should now load automatically
4

Login and Connect Launch Monitor

  1. Open MFE from your Start Menu or desktop shortcut
  2. Login with your TaylorMade credentials
  3. Navigate to Settings
  4. Select your launch monitor brand:
    • Trackman
    • Foresight (GCQuad)
  5. Choose connection method:
    • Ethernet - Best (most stable)
    • USB - Good (reliable)
    • WiFi - Good (ensure strong signal)
  6. Complete the connection process

‚úÖ Installation Complete!

You're ready to start fitting customers with MFE

TROUBLESHOOTING

Quick solutions for common issues

üö´ Launch Monitor Not Connecting?

  1. Open Start Menu and search for MyFittingEXP Connector Service
  2. Launch the Connector Service application
  3. Return to MFE and attempt to connect again
  4. Verify launch monitor is powered on
  5. Check physical connection (Ethernet cable, USB, WiFi signal)
  6. If issue persists, contact TaylorMade IT

‚ùå App Won't Launch or Crashes

  1. Close the app completely (use Task Manager if needed)
  2. Restart your Windows device
  3. Check for Windows updates
  4. Verify Microsoft Store is functioning
  5. Check for MFE app updates in Microsoft Store
  6. Contact TaylorMade IT if problem continues

üîÑ Complete Reinstallation

If you're still experiencing issues after troubleshooting:

  1. Uninstall MFE: Settings ‚Üí Apps ‚Üí Installed Apps ‚Üí Find MFE ‚Üí Uninstall
  2. Uninstall Connector: Control Panel ‚Üí Programs and Features ‚Üí TaylorMade Connector Service ‚Üí Uninstall
  3. Restart your computer
  4. Reinstall both applications using the download links above
