Troubleshooting ‚Äî iPad

Common issues and solutions for internal fitters

üö´ Launch Monitor Not Connecting

  1. Check Wi-Fi network: Go to iPad Settings ‚Üí Wi-Fi. You must be connected to the monitor's Wi-Fi, not your regular network or venue Wi-Fi.
  2. TrackMan: Connect to SSID "TrackMan_####" with password: password
  3. GCQuad: Connect to SSID "GCQuad_####" with password: FSSPORTS
  4. Using Bluetooth? Ensure the device is paired in iOS Bluetooth settings.
  5. Check monitor status: Is it powered on with solid indicator lights (not blinking)?
  6. Restart sequence: Force close MFE, power cycle the monitor, reconnect to monitor Wi-Fi, relaunch MFE.

‚ùå App Crashes on Launch

  1. Check iOS version: Did you recently update iOS? New iOS releases may not be certified yet. Contact IT to verify compatibility.
  2. Force close and relaunch: Swipe up to close the app completely, then reopen.
  3. Check for app updates: Open Company Portal or the install link to see if updates are available.
  4. Reinstall the app: Uninstall and reinstall from my.taylormadegolf.com/myFittingExp/#/tmsupport/ios
  5. ‚ö†Ô∏è Before reinstalling: Ensure all fitting data is synced ‚Äî offline data will be erased on reinstall.

üîë Login Issues

  1. Verify credentials: Use your TaylorMade MFE credentials (usually your email).
  2. Check CAPS LOCK and re-enter password carefully.
  3. Confirm internet: Ensure iPad has internet access (cellular or Wi-Fi to a network with internet).
  4. Account locked? Too many failed attempts will lock the account. Contact TaylorMade IT to unlock, or wait for auto-unlock.
  5. Forgot password? Contact TaylorMade IT for a password reset.

‚öôÔ∏è App Permissions Denied

  1. If you denied Bluetooth, Local Network, or Location permissions on first launch, the app cannot connect to monitors.
  2. Go to iPad Settings ‚Üí Privacy & Security
  3. Find Bluetooth, Local Network, and Location Services
  4. Ensure MFE (myFittingExp) is enabled for each
  5. Restart the MFE app

üìä Data Not Syncing

  1. Were you offline? If the fitting was done without internet, data syncs when you're back online. Open the app with internet to trigger sync.
  2. Check sync status: Look for a sync indicator in the app (spinning icon or Settings ‚Üí Sync).
  3. Verify account: If data shows synced but missing online, confirm you're checking the correct account on the website.
  4. Still missing? Escalate to TaylorMade IT ‚Äî may require server-side investigation.

Troubleshooting ‚Äî Windows

Common issues and solutions for retail partners

üö´ Launch Monitor Not Connecting

  1. Is Connector installed? Check system tray for TaylorMade Connector icon. If missing, download and install.
  2. Is Connector running? If not in system tray, restart MFE (it launches the connector). Or open Services (services.msc) and start "TaylorMade Connector Service".
  3. UAC Approval: Did you click "Yes" on the Windows UAC prompt? If you denied it, uninstall and reinstall the Connector.
  4. Check connection type:
    • USB: Try a different port or cable. Check Device Manager ‚Äî GCQuad should appear under network adapters.
    • Ethernet: Verify cable is connected. PC should get an IP (TrackMan typically 192.168.51.x).
    • Wi-Fi: Ensure PC is on the monitor's Wi-Fi network, not auto-reconnected to another network.
  5. Monitor ready? Indicator lights should be solid (not blinking). Power cycle if needed.
  6. Restart sequence: Close MFE completely, power cycle monitor, relaunch MFE.

‚¨áÔ∏è Can't Install from Microsoft Store

  1. Verify Windows 11: MFE requires Windows 11 (64-bit). Windows 10 is not officially supported.
  2. Store disabled? If your organization disabled Microsoft Store, contact your IT admin to enable it or perform an offline install.
  3. Sign in required: You must be signed into the Store with a Microsoft account to install.
  4. Try direct link: MFE on Microsoft Store
  5. Check internet: Store errors sometimes occur due to connectivity issues.

üîí No Admin Rights for Connector

  1. The Connector installation requires administrator privileges (one-time).
  2. Option 1: Have your IT admin enter credentials at the UAC prompt.
  3. Option 2: Have IT run the Connector installer with "Run as Administrator".
  4. After install: Day-to-day use does not require admin rights.
  5. Still blocked? Contact TaylorMade IT to discuss alternatives.

‚ùå App Won't Launch or Crashes

  1. Windows SmartScreen blocking? If you see "Windows protected your PC", click "More info" ‚Üí "Run anyway".
  2. Check Event Viewer: Windows Logs ‚Üí Application for error entries at the crash time. Note the faulting module for IT.
  3. Recent update? If crashes started after an app update, try reinstalling via Store "Repair" or uninstall/reinstall.
  4. Antivirus blocking? Check if AV is quarantining MFE. Whitelist the app if needed.
  5. Restart PC and try again.

‚è≥ Stuck on "Connecting to Connector Service"

  1. This means the Connector service isn't running properly.
  2. Open Services (Win+R, type "services.msc", Enter)
  3. Find "TaylorMade Connector Service"
  4. Check if it's Started. If not, right-click ‚Üí Start.
  5. If it fails to start, note the error and reinstall the Connector.
  6. Still stuck? Restart PC after Connector install to finalize service registration.

üîë Login Issues

  1. Correct credentials? Retail accounts are different from internal TaylorMade employee accounts.
  2. New user? Check for an activation email from TaylorMade. You may need to activate your account first.
  3. Check CAPS LOCK and re-enter password carefully.
  4. Account locked? Contact TaylorMade IT to unlock (or wait for auto-unlock).
  5. Server error with correct credentials? Escalate to Tier 2 ‚Äî possible backend issue.

üîå Connector Service Problems

  1. Not installed: Download here, run installer, click "Yes" on UAC.
  2. Won't start: Open Services (services.msc), find "TaylorMade Connector Service", right-click ‚Üí Start.
  3. Crashes on start: Note any error, then uninstall from Control Panel ‚Üí Programs, reboot, reinstall.
  4. Denied UAC earlier: Uninstall Connector, reinstall, and click "Yes" this time.
  5. Check firewall: Ensure the Connector is allowed through Windows Firewall.

üîÑ Complete Reinstallation

If issues persist after other troubleshooting:

  1. Uninstall MFE: Settings ‚Üí Apps ‚Üí Installed Apps ‚Üí Find MFE ‚Üí Uninstall
  2. Uninstall Connector: Control Panel ‚Üí Programs and Features ‚Üí TaylorMade Connector Service ‚Üí Uninstall
  3. Reboot the PC
  4. Reinstall MFE: Microsoft Store
  5. Reinstall Connector: Let MFE prompt you, or download directly

üìä Data Not Syncing

  1. Check internet: Can you browse websites? MFE needs internet for cloud sync.
  2. Check sync status: Look in app Settings ‚Üí Sync for pending data indicator.
  3. Firewall: Ensure *.taylormadegolf.com is allowed. Try a mobile hotspot as a test.
  4. Verify account: If synced but not showing on website, confirm you're checking the correct account.
  5. Data loss suspected? Escalate to TaylorMade IT immediately for server-side investigation.
